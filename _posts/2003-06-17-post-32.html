Ben's Mac Software MouseZoom is a Mac OS X Cocoa pref pane which speeds up your mouse cursor movement in Mac OS X. As of version 2.0, the speed is set "live" without having to log out before it takes effect. To do this, MouseZoom uses the same method as Apple's mouse panel.
自分はAppleの標準のマウス速度でやってたのですが、それだと画面が大きいとなんか遅い。画面の端から端までかなりかかる(笑) ということで、昔はAppleUSBMouse Turbo Edition っての使ってたんですがいつのまにやら配布されなくなり、どこにもなくなってしまったのです。 そしたらこれを見つけました。 なかなかいい感じです。上記のようなお悩みがある方には是非使ってもらえればと思います。フリーだしねぇ〜。
