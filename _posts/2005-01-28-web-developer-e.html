Web Developer Extension on chrispederick.com The Web Developer extension for Mozilla Firefox and Mozilla adds a menu and a toolbar to the browser with various web developer tools.
昨日からFirefoxを使い始めたのだが、何点か今まで使用していた ブラウザの慣れからくる操作感の違いはあるものの、意外といい感じである。 例の如くExtensionを探しまくっていたが、表題の「Web Developer Extension」はなかなかいい感じ。 Webの開発している人にはちょっとしたときに使える機能があったりする。 たとえば、画面内のフォーム等のいろいろな要素を操作したり表示したり、セッションやクッキーの操作を行なったり。
